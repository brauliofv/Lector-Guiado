# Lector Guiado - Chrome Extension for Guided Reading

A simple yet powerful browser extension designed to improve reading speed and focus through a customizable visual guide.

## About The Project

Lector Guiado (Guided Reader) helps users train their reading speed and maintain concentration on long texts. By selecting a piece of text on any webpage, the extension overlays a visual guide (a dot) that moves across the lines at a configurable speed in Words Per Minute (WPM), guiding the reader's eyes and reducing regressions (the act of re-reading words).

## ‚ú® Features

*   **Animated Visual Guide:** A dot moves smoothly over the selected text, following every line.
*   **Customizable Speed:** Adjust the reading speed from 100 to 600 Words Per Minute (WPM) from the popup menu.
*   **Intuitive Activation:** Simply select text, right-click, and choose "Start Lector Guiado".
*   **Full Control:** Stop the animation at any time with a single click anywhere on the page.
*   **Clean Experience:** The text selection is automatically cleared for a distraction-free reading experience.
